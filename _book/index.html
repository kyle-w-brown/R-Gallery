Chapter 1 Introduction

Welcome the R Gallery Book, a complete guide to the R Graph Gallery website. This information is taken directly from R graph gallery with careful detail in reproducing plots and completing ideas. This material created by Kyle W. Brown as way to have one single reading collection of updated R gallery plots and graphs.



While this book was created to encapsulate the entire R Graph Gallery website into one readable source, another purpose is serving as introductory level into data visualization using R programming language.
